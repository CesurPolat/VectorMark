Advanced Features & Testing

        
LanceDB test suite and advanced query capabilities.

        
        

          
Run Test Suite

          
Execute comprehensive LanceDB integration tests in the browser console:

          
            window.runLanceDBTests()
          
          
Tests include: initialization, document insertion, vector search, metadata filtering, and store cleanup.

        


        

          
Import Test Runner

          
For manual testing, import the test runner module in browser console:

          
            import { runLanceDBTests } from './test_documents/lancedbTestRunner.js'
          
          
Then call: runLanceDBTests()

        


        

          
About LanceDB

          
    
            
  • ✓ Advanced vector similarity search with configurable distance metrics
    • 
            
  • ✓ SQL WHERE clauses for metadata filtering (status, date range, tags)
    • 
            
  • ✓ WASM-based execution in browser environment (Extension-compatible)
    • 
            
  • ✓ IndexedDB or OPFS backend (auto-detected for maximum compatibility)
    • 
            
  • ✓ 384-dimensional embeddings from all-MiniLM-L6-v2 model
    •

            The test database is isolated and does not affect the main VectorMark application.
